# Angular Rich Text Editor — Web Spell Checker Sample

This repository is a minimal Angular sample showing integration of Web Spell Checker with the EJ2 [Angular Rich Text Editor](https://www.syncfusion.com/angular-components/angular-rich-text-editor).

## Quick start

Prerequisites:

- Node.js 20.x (recommended) and Also supported: Node.js 22.x or 24.x
- npm

Install dependencies and run the app:

```bash
npm install
ng serve
```

Open http://localhost:4200/ in your browser.

## What this sample shows

- How to include the Rich Text Editor in an Angular component
- How to enable/configure the Web Spell Checker integration (see Integration UG link for exact steps)
