# Angular Rich Text Editor — Web Spell Checker Sample

This repository is a minimal Angular sample showing integration of Web Spell Checker with the EJ2 Angular Rich Text Editor.

## Quick start

Prerequisites:

- Node.js (16+) and `npm`

Install dependencies and run the app:

```bash
npm install
ng serve
```

Open http://localhost:4200/ in your browser.

## What this sample shows

- How to include the Rich Text Editor in an Angular component
- How to enable/configure the Web Spell Checker integration (see Integration UG link for exact steps)
